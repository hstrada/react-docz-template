## 📦️ How to install the project

To clone the project, use the commands bellow:

```bash
  # Clone the repository
  ❯ git clone git@github.com:hstrada/react-docz-template.git

  # Enter directory
  ❯ cd worldz
```

### FrontEnd

```bash
  # Enter frontend directory
  > cd worldz

  # Install the dependencies
  ❯ yarn
  
  # Run the Project
  ❯ yarn start
```

**Run Docz**

```bash
  # Enter frontend directory
  > cd worldz

  # Start the project
  ❯ yarn docz:dev
```
